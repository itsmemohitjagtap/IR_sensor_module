ğŸ”§ **How It Works**

- The IR LED emits infrared light which reflects off nearby objects.  
- The IR receiver detects the reflected signal.  
- Signal conditioning and comparison are performed using an op-amp.  
- The digital output triggers when the signal crosses a set threshold, while the analog output provides raw signal variation.  
- Sensitivity can be adjusted using the onboard trimmer potentiometer.
